Roblox Macros Launcher
A lightweight, Python-based graphical user interface designed to streamline the management and execution of Roblox (via Bloxstrap) and AutoHotkey scripts. Built using CustomTkinter, this application serves as a centralized desktop hub for users who frequently utilize macros.

Features
Automated First-Time Setup: Prompts the user to define the file paths for their Bloxstrap executable and AutoHotkey scripts directory upon initial launch.

Dynamic Script Loading: Automatically scans the designated directory for .ahk files and generates a scrollable list of interactive launch buttons.

Persistent Configuration: Saves user preferences securely to the Windows AppData/Local directory, avoiding system permission conflicts and ensuring seamless subsequent launches.

Modern Interface: Utilizes a dark-mode graphical user interface designed for a clean and unobtrusive desktop footprint.

Prerequisites
To run or compile the source code, the following are required on the host system:

Python 3.x

CustomTkinter library

AutoHotkey (to execute .ahk files)

Bloxstrap
